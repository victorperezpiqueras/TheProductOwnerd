TheProductOwnerd is a web application focused on giving the Product Owner the tools to successfully manage software projects using an agile methodology.

Using this application the Product Owner will be able to:
  • calendar_view_day Manage the Product Backlog of a project
  • group Invite new members to collaborate in the Scrum team.
  • bar_chart Control the Product Burndown Chart and project specific metrics such as velocity
  • multiline_chart Perform forecasting of the project and team performance
